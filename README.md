===========

Simple tool to convert image to table html.


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'image2table'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install image2table

## Usage

```
image2table -i image_path -o html output
```
